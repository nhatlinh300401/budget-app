# 💰 Spendwise – Budget Tracking App

A mobile budget tracking app built with React Native and Expo.
Users can track income and expenses, view financial summaries, and visualize spending by category.

---

## 🚀 Features

* Add and delete transactions
* Categorize expenses (Food, Transport, Shopping, etc.)
* View income, expense, and balance summary
* Persistent local storage using AsyncStorage
* Interactive pie chart for expense breakdown
* Clean and modern UI
* Confirmation dialog for safe deletion

---

## 🛠 Tech Stack

* React Native (Expo)
* TypeScript
* Context API (state management)
* AsyncStorage (data persistence)
* react-native-chart-kit (data visualization)
